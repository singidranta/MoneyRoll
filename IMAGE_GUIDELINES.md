# MoneyRoll Image Guidelines

## Генерация изображений

**ВСЕГДА используй зелёный экран (chroma key) для генерации:**

```
Product photo on green screen background (chroma key). [описание предмета]. Clean flat style icon for game UI.
```

## Категории изображений

### 1. Еда
- Все изображения PNG в `/assets/props/flat/food/`
- Белый/прозрачный фон на зелёном экране
- Названия: shawarma.png, energy.png, hotdog.png, sushi.png, pizza.png, salad.png, ramen.png, steak.png

### 2. Иконки сортировки
- Все изображения PNG в `/assets/icons/`
- Формат: sort-[category].png
- Категории: plastic (жёлтый), glass (зелёный), paper (синий), metal (серый)

### 3. Одежда
- Все изображения PNG в `/assets/props/flat/clothing/`
- Названия: cap.png, beanie.png, hoodie.png, jeans.png, boots.png
- Должны быть видны на персонаже

### 4. Предметы для сортировки
- Все изображения PNG в `/assets/props/flat/trash/`
- Должны быть узнаваемы по категории

## Цветовые коды категорий
- Пластик: #facc15 (жёлтый)
- Стекло: #22c55e (зелёный)
- Бумага: #3b82f6 (синий)
- Металл: #a1a1aa (серый)
